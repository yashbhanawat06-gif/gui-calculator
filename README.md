This project is basically build for performing arithmetic operations like addition, subtraction, etc.
It is built using Python and Tkinter library.

FEATURES:
  Addition,
  Subtraction,
  Multiplication,
  Division,
  Square of a number,
  Percentage,
  Backspace,
  Clear All,
  Decimal,

HOW TO RUN?
  Download the project>
  Open terminal in the folder>
  Run: python calculator.py
  
OR YOU CAN DIRECTLY COPY THE CODE AND PASTE IT IN VS CODE AS WELL
